Sobre a Aplicação

A aplicação foi criada para facilitar o acompanhamento e a geração de pagamentos Pix recorrentes, consolidando todas as cobranças mensais em um único local. Com valores pré-definidos para cada mês, os pagamentos realizados atualizam automaticamente o banco de dados da aplicação por meio de webhooks fornecidos pelo banco EFI.

Java Spring (backend)

  • Tecnologia: Desenvolvido em Java Spring, oferecendo robustez e leveza.
  • Segurança: Dados sensíveis, como senhas, são armazenados de forma segura com criptografia BCrypt, em conformidade com as exigências da LGPD.
  • DTOs: As respostas às requisições utilizam DTOs (Data Transfer Objects) para proteger dados internos e evitar exposição desnecessária.
  • Gerenciamento de Sessões: Sessões de usuário são gerenciadas por tokens JWT, garantindo segurança e desempenho.
  • Integração com Webhooks: Atualização automatizada do status de pagamentos por meio de notificações do banco.
Mais detalhes no repositório: pagamento-carnes-spring

AngularJs(frontend)

  • Tecnologia: Desenvolvido com Angular, conhecido por sua leveza e confiabilidade.
  • Segurança: Todas as chamadas à API utilizam o protocolo HTTPS (TLS), garantindo criptografia ponta-a-ponta.
  • Interface: SPA (Single Page Application) fluida e responsiva, projetada para facilitar a gestão dos pagamentos independentemente da plataforma utilizada.
